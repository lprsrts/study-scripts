# Study Scripts

A collection of helpful Python scripts for various study topics, including physics, mathematics, and data visualization.

## Contents

This repository contains visualization and computational scripts for:

- **Physics**: Electric field calculations, charge distributions
- **Statistics**: Central Limit Theorem demonstrations
- **Mathematics**: Various mathematical concepts and visualizations

## Scripts

- `CurlIdentityOfElectricField.py` - Visualization of electric field curl identities
- `GaussianSmoothedChargeDist.py` - Gaussian smoothed charge distribution calculations
- `ZentralerGrenzwert.py` - Central Limit Theorem (Zentraler Grenzwertsatz) demonstration

## Requirements

```bash
pip install -r requirements.txt
```

## Usage

Run any script directly with Python:

```bash
python script_name.py
```

Output images are saved to the `output_images/` directory.
